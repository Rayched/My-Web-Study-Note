HTML이란 무엇인가?


HTML(Hypertext Markup Language)란 웹 페이지의 콘텐츠를 정의하기 위해서 사용하는 언어로
쉽게 말하자면, 웹 페이지의 뼈대를 만든다고 생각하면 될 것이다.
HTML은 프로그래밍 언어가 아닌, 우리가 보는 웹 페이지가 어떤 구조로 되어있는지를
브라우저에게 알려주는 마크업 언어이다.
HTML은 Element, 요소, 태그들로 구성되어 있으며
HTML에서 사용되는 각 태그들은 웹 페이지의 여러 콘텐츠를 감싸고 있거나
아니면 현재 문서에서 다른 문서로 이동하는 하이퍼링크를 만들거나
특정 단어를 강조하는 등의 다양한 역할을 하고있다.
